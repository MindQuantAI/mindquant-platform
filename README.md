# MindQuant

AI-powered Market Intelligence Platform focused on behavioral sentiment modeling and narrative analysis.

MindQuant analyzes market attention, social sentiment, volatility regimes, and psychological tone to detect shifts that often precede structural market changes.

---

## Core Components

- Sentiment Signal Index (0–100 unified scale)
- Volatility Regime Detection
- Risk Pulse Modeling
- Narrative Cluster Tracking
- Behavioral Tone Classification

---

## Research Philosophy

Markets reorganize psychologically before they reorganize technically.

MindQuant focuses on identifying narrative pressure, bias compression, and sentiment inflection before price confirms structural shifts.

Signal before noise.

## Technical Architecture

MindQuant is built as a modular AI-first SaaS platform.

### Stack

- **Frontend:** React 18, TypeScript, Vite, Tailwind, shadcn/ui
- **State Management:** TanStack Query, Zustand, React Context
- **Backend:** Supabase (Postgres + Edge Functions)
- **AI Layer:** OpenAI GPT-4 / 4o-mini via AI Gateway
- **Payments:** Stripe (Checkout, Webhooks, Portal)
- **Animation:** Framer Motion

---

## System Architecture

Landing / Auth
